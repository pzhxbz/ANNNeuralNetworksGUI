#使用c#给手写识别写了个GUI

##使用说明

* 在那一个黑色的框内写下你的数字，记住写大一点，因为会压缩成为大小为8*8的图片作为输入
* 之后点击开始识别，神经网络会给出一个输出，显示在右上角
* 如果正确，请点击正确按钮；如果错误，请选择正确的答案，然后点击改正。

如果不能运行请安装.NetFramework
